SbtDeskTran version changes

Keep release/change details here instead of putting them in the release body.

Last released: v2026.07.13.36

-----------------------------------------------------------------
v2026.07.13 - Find in Translate/Notes and safer auto-update
-----------------------------------------------------------------

App:
- Added the Diff-style find bar to Translate and Notes, including Ctrl+F,
  previous/next navigation, case-sensitive, whole-word, and regex options.
- Translate find searches both Source and Translated panels.
- Notes find searches the current note body and refreshes when switching notes.
- Compact mode hides the search/find controls in Diff, Translate, and Notes.
- Cleaned up the update temp files.

Auto Update:
- Downloads and extracts the new exe, stages it inside the app folder, then
  cleans temp files before showing Restart.
- Restart now only renames the staged exe into place and starts the app from
  its own folder. UNC app folders use PowerShell when CMD cannot enter them.
- Checks the PyInstaller archive and SHA-256 before replacing the app.

-----------------------------------------------------------------
v2026.07.10 - Faster translation, network fallback, and auto-update fixes
-----------------------------------------------------------------

Network module:
- Added proxy detection, opener builder, HTTP request functions.
- Added WinINet fallback for proxy/PAC and translation.

Translate:
- Translate now updates smarter after you edit text. Unchanged parts are kept,
  deleted parts are removed, and only new or edited parts are translated again.
- This makes auto-translate faster and avoids re-translating the whole text
  after small changes.

Auto Update:
- Fixed a self-update hang caused by a stuck find.exe window. The updater now
  replaces the exe by retrying file move/copy directly.
- Restart after update now changes into the app folder and starts the exe by
  file name, matching the same launch context as running it manually from CMD.
  If the app folder cannot be entered by CMD, the helper falls back to
  PowerShell Start-Process with the app folder as WorkingDirectory.
- The update helper now runs hidden, so timeout/retry commands no longer show
  a CMD window during self-update.

-----------------------------------------------------------------
v2026.07.09 - Persistent tab UI, auto-translate on paste, Diff pane stability, Update dialog UI
-----------------------------------------------------------------

App:
- Tab switching now keeps each tab UI alive and only hides/shows cached
  tab frames, preserving pane sizes, cursor/selection, scroll position,
  and undo state during normal tab changes.
- Compact, word-wrap, theme, and Translate layout changes still rebuild
  the affected tab UI intentionally so their controls and layout update
  correctly.
- Added <<Paste>> event binding to source text to catch both right-click
  and Ctrl+V paste, ensuring auto-translate triggers on any paste method.
- Preserve pending source text when new input arrives while a previous
  translation is still in progress. The pending text is automatically
  re-translated once the current translation finishes, covering both
  overwrite-paste and insert-paste during active translation.

Diff:
- Prevented Diff pane sash settings from being saved/restored while the
  PanedWindow is hidden or still reporting a temporary layout width.
- Added recovery for previously corrupted Diff left-pane width/ratio
  settings that pinned the left side near its minimum width on remote PCs.

Auto Update:
- Fixed GitHub release asset matching for versioned packages named like
  SbtDeskTran-2026.07.07.8.zip.
- Fixed PowerShell fallback download for binary release assets. The fallback
  now streams bytes and Base64-decodes them in Python instead of writing the
  string "System.Byte[]" into the downloaded zip.
- Added update package validation before replacing the running executable:
  downloaded .zip files must be valid zip archives, the archive must contain
  SbtDeskTran.exe, and the extracted executable must have a valid Windows
  MZ header.
- Prevented invalid HTML/error/proxy responses or malformed downloads from
  being copied over SbtDeskTran.exe during self-update.
- Self-update now waits for the old process to exit, verifies the copied exe
  size, delays briefly before restart, and cleans staged update files after a
  successful replacement.
- Restart now launches the update helper detached from the app process and
  exits the old app process immediately after saving state, preventing the old
  process from staying grouped with the restarted app in Task Manager.
- The update helper now starts the new exe from the running app folder after
  copy/cleanup/delay, using pushd so UNC/network app folders get a valid
  working directory before launch.
- If the old app process does not exit after Restart, the helper force-stops
  that PID before replacing the exe.
- Replaced netsh/PowerShell network fallback with in-process WinINet proxy/PAC
  fallback, so startup checks, manual Check Update, translation, and update
  downloads can still use Windows proxy/PAC without creating a child process
  that Task Manager groups as SbtDeskTran (2).
- The app window icon is applied from the bundled icon resource, and the CI
  PyInstaller build now bundles icon.ico so the title bar can use the same
  icon as the executable.

v2026.07.08 — Editable diff, auto-save notes, large translate chunks, CI pipeline
-----------------------------------------------------------------

New features:
- VirtualDiffEditor: fully editable side-by-side diff view with
  undo/redo, find/replace, minimap, detail panel, word-wrap toggle,
  synchronized scrolling.
- Note Auto Save with dirty-state indicator (✓ Saved / ● Unsaved).
- Self-updater module (checks GitHub releases, replaces .exe).
- CI release helpers (version bump, changelog verification, asset
  packaging, GitHub Release creation with release notes).
- Placeholder stipple pattern for diff empty lines.

Diff Engine:
- Refined tokenizer: word chars and punctuation now split correctly
  for more precise inline diffs.
- Character-level diff for non-CJK replacement tokens when word
  diff is off.
- Whitespace-ignoring diff mode.

Translator:
- Auto-split large texts (>700 chars) into chunks to bypass Google
  Translate length limits.
- Translation options (engine, languages) now persist immediately.

Widgets:
- DiffViewer: replaced fixed grid with resizable PanedWindow (sash).
- Synchronized horizontal scrolling between left/right panes.
- Snapshot/restore API for widget state preservation.

App:
- Status bar shows per-panel metrics (chars, line, column,
  selection length).
- Font size tracked per-tab (translate, diff, note).
- Compact/layout/theme/wrap toggles capture widget state before
  rebuilding UI (no data loss on toggle).
- Note delete now shows confirmation dialog.
- Note save auto-creates new note if none selected.
- Update check runs 1.5s after startup.
