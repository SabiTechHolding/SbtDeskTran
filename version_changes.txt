SbtDeskTran version changes

Keep release/change details here instead of putting them in the release body.

Last released: v2026.07.09.19

----------------------------------------------------------------------
v2026.07.09 - Persistent tab UI, auto-translate on paste, Diff pane stability
----------------------------------------------------------------------

App:
- Tab switching now keeps each tab UI alive and only hides/shows cached
  tab frames, preserving pane sizes, cursor/selection, scroll position,
  and undo state during normal tab changes.
- Compact, word-wrap, theme, and Translate layout changes still rebuild
  the affected tab UI intentionally so their controls and layout update
  correctly.
- Added <<Paste>> event binding to source text to catch both right-click
  and Ctrl+V paste, ensuring auto-translate triggers on any paste method.
- Preserve pending source text when new input arrives while a previous
  translation is still in progress. The pending text is automatically
  re-translated once the current translation finishes, covering both
  overwrite-paste and insert-paste during active translation.

Diff:
- Prevented Diff pane sash settings from being saved/restored while the
  PanedWindow is hidden or still reporting a temporary layout width.
- Added recovery for previously corrupted Diff left-pane width/ratio
  settings that pinned the left side near its minimum width on remote PCs.

Auto Update:
- Fixed GitHub release asset matching for versioned packages named like
  SbtDeskTran-2026.07.07.8.zip.
- Fixed PowerShell fallback download for binary release assets. The fallback
  now streams bytes and Base64-decodes them in Python instead of writing the
  string "System.Byte[]" into the downloaded zip.
- Added update package validation before replacing the running executable:
  downloaded .zip files must be valid zip archives, the archive must contain
  SbtDeskTran.exe, and the extracted executable must have a valid Windows
  MZ header.
- Prevented invalid HTML/error/proxy responses or malformed downloads from
  being copied over SbtDeskTran.exe during self-update.

----------------------------------------------------------------------
v2026.07.08 — Editable diff, auto-save notes, large translate chunks, CI pipeline
----------------------------------------------------------------------

New features:
- VirtualDiffEditor: fully editable side-by-side diff view with
  undo/redo, find/replace, minimap, detail panel, word-wrap toggle,
  synchronized scrolling.
- Note Auto Save with dirty-state indicator (✓ Saved / ● Unsaved).
- Self-updater module (checks GitHub releases, replaces .exe).
- CI release helpers (version bump, changelog verification, asset
  packaging, GitHub Release creation with release notes).
- Placeholder stipple pattern for diff empty lines.

Diff Engine:
- Refined tokenizer: word chars and punctuation now split correctly
  for more precise inline diffs.
- Character-level diff for non-CJK replacement tokens when word
  diff is off.
- Whitespace-ignoring diff mode.

Translator:
- Auto-split large texts (>700 chars) into chunks to bypass Google
  Translate length limits.
- Translation options (engine, languages) now persist immediately.

Widgets:
- DiffViewer: replaced fixed grid with resizable PanedWindow (sash).
- Synchronized horizontal scrolling between left/right panes.
- Snapshot/restore API for widget state preservation.

App:
- Status bar shows per-panel metrics (chars, line, column,
  selection length).
- Font size tracked per-tab (translate, diff, note).
- Compact/layout/theme/wrap toggles capture widget state before
  rebuilding UI (no data loss on toggle).
- Note delete now shows confirmation dialog.
- Note save auto-creates new note if none selected.
- Update check runs 1.5s after startup.
